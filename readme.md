## ✨ Features

- **Mega Menu Navbar** — Desktop dropdown with featured properties, property types, popular locations, and resource cards
- **Hero Section** — Full-screen background with an advanced property search form (location, type, price, beds, baths, sqft)
- **Featured Properties** — Card grid with image zoom, badges, photo count, and agent info
- **Our Services Tabs** — Tabbed interface covering Buying, Selling, Renting, Investing, and Commercial real estate
- **Client Testimonials** — Bootstrap Carousel with custom indicators and navigation
- **FAQ Accordion** — Collapsible answers to common real estate questions
- **Contact Section** — Contact info cards + full message form with validation
- **Responsive Design** — Mobile-first layout with custom media queries
- **Scroll-to-top Button** — Fixed position scroll shortcut
- **Custom Animations** — Smooth transitions on hover, navbar toggler icon swap, image zoom

## 🚀 Getting Started

No build tools or package managers required — this is a pure HTML/CSS/JS project.

### 1. Clone the repository

```bash
git clone https://github.com/your-username/EliteHomes.git
cd EliteHomes
```

### 2. Open in browser

Just open `index.html` directly in your browser, or use a local dev server:

```bash
# Using VS Code Live Server extension (recommended)
# Right-click index.html → Open with Live Server

# Or using Python
python -m http.server 5500
```

Then visit `http://localhost:5500`

## 🎨 Design System

| Token                 | Value     | Usage                  |
| --------------------- | --------- | ---------------------- |
| `--primary-purple`    | `#6b21a8` | Main brand color       |
| `--link-hover-purple` | `#7f22fe` | Nav links, CTAs        |
| `--light-purple`      | `#8b5cf6` | Accents, badges        |
| `--dark-purple`       | `#5b21b6` | Hover states           |
| `--accent-green`      | `#10b981` | Success states         |
| `--text-dark`         | `#1f2937` | Body text              |
| `--text-light`        | `#6b7280` | Muted / secondary text |

**Font:** [Exo](https://fonts.google.com/specimen/Exo) via Google Fonts

## 📱 Responsive Breakpoints

| Breakpoint       | Width      | Notes                                |
| ---------------- | ---------- | ------------------------------------ |
| Mobile           | `< 992px`  | Collapsed navbar, stacked form       |
| Tablet / Desktop | `≥ 992px`  | Mega menu enabled, form at 75% width |
| Large Desktop    | `≥ 1200px` | Mega menu centered with wider width  |
